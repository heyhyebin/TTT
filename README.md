# Grain TTT Prediction

본 프로젝트는 합금 성분 데이터를 기반으로  
TTT(Time-Temperature-Transformation) 선도를 예측하기 위한  
데이터셋 구축 및 인공지능 모델 개발을 목표로 한다.

데이터 전처리, TTT 데이터 정량화, 그리고  
AI 학습용 데이터셋 생성 과정을 포함한다.

---

## 데이터 처리 과정

데이터셋 구축은 다음 두 단계로 진행된다.

### 1. 합금 성분 데이터 전처리
파일: `ttt_dataset_preprocessing.py`

이 스크립트는 합금 성분 데이터를 정리하는 역할을 한다.

주요 기능:
- 문자열 데이터를 숫자 데이터로 변환  
  (예: `"0.38-0.43"` → 평균값)
- `"max. 0.035"` 형태의 데이터 처리
- 결측값(`-`, NaN) 처리
- 주요 합금 원소 데이터 추출

정리된 합금 데이터는 이후 데이터셋 생성 단계에서 사용된다.

---

### 2. TTT 데이터 특징 추출 (확장 코드)
파일: `ttt_dataset_preprocessing_extend.py`

이 확장 코드는 TTT 선도 데이터를 분석하여  
변태 관련 주요 정보를 추출한다.

주요 기능:
- TTT 선도의 point 데이터 읽기
- 변태 속도가 가장 빠른 **nose point** 탐색
- 변태 시작(start) / 완료(finish) 정보 추출
- 합금 성분 데이터와 TTT 정보를 결합
- AI 학습용 데이터셋 생성

이 과정을 통해 그래프 형태의 TTT 데이터를  
AI 모델 학습에 사용할 수 있는 **수치 데이터로 변환**한다.

---

## Notes

현재 데이터셋에서는 주요 합금 원소만 사용하여 데이터를 구성하였다.

사용된 주요 원소:
- C
- Mn
- Si
- Ni
- Cr
- Mo
- V
- Cu

TTT 변태 거동에 영향을 줄 수 있는 다른 원소들
(예: Al, Nb, Ti, Co, W 등)을 포함할지에 대해서는
현재 팀 내에서 검토 중이다.

전체 원소를 사용할 경우 더 많은 합금 정보를 반영할 수 있지만,
결측 데이터 증가 및 모델 복잡도가 높아질 가능성이 있다.

따라서 본 프로젝트에서는
1차적으로 주요 원소 기반 모델을 구축한 후
필요 시 추가 원소를 포함한 확장 모델을 고려할 예정이다.
